# Precision Construction Services Mock-Up

Static homepage mock-up and strategy notes for a proposed rebuild of Precision Construction Services.

## Files

- `index.html` - browser-ready homepage mock-up for Vercel/static hosting
- `precision-construction-audit-and-rebuild-plan.md` - website audit, rebuild strategy, conversion plan, and pitch framing

## Launch

This repo can be deployed directly to Vercel as a static site. The homepage is `index.html` at the project root.
